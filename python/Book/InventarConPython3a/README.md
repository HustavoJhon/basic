### Capitulo 5 "El Chiste"

- **Caracteres de Escape**

|Caracteres de Escape|Lo que imprime|
|--|--|
| \\  | Barra inversa(\)  |
| \'  | Comillas simple(')|
| \"  | Comilla doble(")  |
| \n  | Salto de linea    |
| \t  | Tabulador         |
